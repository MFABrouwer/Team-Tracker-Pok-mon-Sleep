# 🌙 Pokémon Sleep — Team Tracker & Weekplanner

A personal Excel-based tracker for **Pokémon Sleep**, built to evaluate, score, and organize your Pokémon roster across multiple team roles.

> **Free to use for any Pokémon Sleep player!**
> The Excel file uses Dutch functions and interface labels, but all in-game content (Pokémon names, berries, ingredients, skills, natures) is kept in **English** — so it works for any player regardless of language.

---

## 📋 What's inside

The workbook contains four sheets:

### 1. `Data` — Main Roster Tracker
The core of the tracker. For each Pokémon you enter:
- Name, level, and type
- Berry and ingredient outputs (at level 1, 30, and 60)
- Skill progression (levels 10 through 100)
- Nature, with positive and negative effects
- Scores for Berry, Ingredient, and Skill performance
- Team role assignment
- A calculated **Total Score** with an automatic recommendation:
  - ⭐ Keep
  - 📈 Level Up
  - 🤔 Bench
  - ❌ Transfer
- Notes field for evolution tips and team synergy advice

### 2. `Weekplanner` — Weekly Overview
A compact view of your active roster, useful for planning your weekly team composition and priorities.

### 3. `Referentie` — Reference Table
A lookup sheet containing:
- Type → Berry mappings
- Full Nature table with positive/negative effects
- Complete Pokédex list for quick reference

### 4. `Helper_Ingredient` — Ingredient Guide
Ingredients sorted by cooking category:
- Curries/Stews
- Salads
- Desserts/Drinks

---

## 🛠️ Requirements

- Microsoft Excel (recommended) or LibreOffice Calc
- The file uses **Dutch Excel function names** internally — if you open it in a non-Dutch Excel installation, formulas will automatically convert to your local language equivalent

---

## 📝 Notes

- Conditional formatting is applied throughout for quick visual scanning
- The scoring system is based on community-established role weights for Berry, Ingredient, and Skill team compositions
- Evolution tips and team advice in the Notes column are based on general Pokémon Sleep meta knowledge at the time of building
